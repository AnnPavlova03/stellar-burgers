# Stellar-Burger

## Описание проекта
Сервис по заказу космических бургеров, где каждый может создать свой уникальный бургер, выбрав булки, начинку и соус по своему инопланетному вкусу.

## В проекте реализовано:
- Процесс заказа: Пользователи могут составить свой бургер, добавляя ингредиенты в конструктор, удалять их, оформлять заказы и выбирать способы оплаты.
- Авторизация пользователя: В личном кабинете пользователи могут просматривать свои заказы.
- Страница с лентой заказов: Возможность отслеживать количество выполненных заказов.

## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```
## Сборка

```
npm run build
```

или

```
yarn build
```
